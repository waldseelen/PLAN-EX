# Legacy TO-DO Arşivi

Bu klasör, Plan.Ex projesine birleştirilmeden önceki TO-DO uygulamasının proje dosyalarını içerir.

## Durum

TO-DO uygulaması Plan.Ex ana projesine başarıyla entegre edilmiştir:

- **Layout ve UI**: TO-DO'nun dark-tech tasarımı Plan.Ex'e taşındı
- **Renk Paleti**: Elektrik cyan (#00aeef) ve altın gold (#ffd200) tema
- **Bileşenler**: Header, RightPanel, Sidebar bileşenleri adapte edildi
- **Stiller**: Glass panel, card-tech, circuit pattern efektleri eklendi

## Arşivlenen Dosyalar

- `package.json` - Bağımlılık listesi
- `vite.config.ts` - Vite yapılandırması
- `tsconfig.json` - TypeScript ayarları
- `tailwind.config.js` - Tailwind tema
- `Dockerfile` / `nginx.conf` - Deployment dosyaları
- Dokümantasyon dosyaları

## Kaynak Kod

TO-DO'nun kaynak kodu `TO-DO/src/` klasöründe referans olarak korunmaktadır.
